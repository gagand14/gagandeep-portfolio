Inventory Management Tool

Excel VBA Power Automate SQL Automation

An automated inventory tracking system that streamlines asset management, reduces manual data entry, and provides real-time visibility into hardware inventory.

Project Overview

Developed to address the challenge of tracking hundreds of IT assets across multiple data center locations. The tool automates inventory updates, generates reports, and alerts stakeholders when assets require maintenance or replacement.

Key Features

  • ▸ Automated asset tracking with barcode scanning integration
  • ▸ Power Automate workflows for automatic inventory updates from procurement systems
  • ▸ Excel-based dashboard with VBA macros for data validation and reporting
  • ▸ SQL database backend for centralized data storage and multi-user access
  • ▸ Automated email notifications for warranty expirations and maintenance schedules

Technical Implementation

The system consists of three main components: an Excel front-end with custom VBA forms for data entry, Power Automate flows that sync data from external systems, and a SQL Server database for persistent storage. The integration between components uses REST APIs and ODBC connections for real-time data synchronization.

Problem Solved

Prior to this tool, inventory management was done manually through spreadsheets, leading to frequent data entry errors, outdated records, and difficulty in locating assets. The solution reduced inventory discrepancies by 80% and saved approximately 15 hours per week in manual data entry work.

Outcomes & Impact

  • ✓ Reduced inventory tracking errors by 80% through automation
  • ✓ Saved 15+ hours weekly by eliminating manual data entry tasks
  • ✓ Improved asset lifecycle management with proactive maintenance alerts
  • ✓ Enabled real-time visibility into inventory across multiple locations

Technologies Used

  • • Microsoft Excel with VBA
  • • Power Automate (Microsoft Flow)
  • • SQL Server Database
  • • REST APIs
  • • ODBC Connections

Key Learnings

  • • API integration patterns
  • • Database design and optimization
  • • User interface design for non-technical users
  • • Change management in IT operations
